Convert computer data units to each other.

== Description ==

With using Bits2Bytes users can convert units from bits to petabytes, to each other, Bits2Bytes use ajax and no page reloaded for convert units.
You simply add Bits2Bytes widget to your site or blog widget area, and users can convert units.useful and simple plugin for computer science student or other users.
if you love this addone , give it a good rating on plugin page or if you see plugin correctly work for you then please click "Works" on plugin page.
if you found any bug, issue or need support please don't hesitate to [create a support ticket.](https://wordpress.org/support/plugin/bits2bytes)

== Installation ==
1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to "Appearance > Widgets" and use "Bits2Bytes Widget"
